Interacting with the SMS gateway

All of this game's administration will happen over SMS (barring a cataclysmic system malfunction).

The SMS gateway for this game is: +16178703381.

Commands

Command Description
join <ldap> <alias>

Used to join the game.

  • ldap: your @google.com ldap; given to your assassin to track you.
  • alias: a secret alias for use on the scoreboard and activity feed.
kill <secret>

Used to record a kill of your target.

  • secret: your target's secret code that they will relinquish upon death.
secret

Used to retrieve a forgotten secret code.
target

Used to retrieve a forgotten target.
quit <secret>

Used to quit the game. Warning: this command does not require confirmation and is difficult to reverse. Only send it if you really intend to leave the game.

  • secret: your secret code.

Note on secret codes

The secret code was designed as a stateless way to ensure both parties agree that a kill has occured. To prevent brute-forcing by the killer, the killer can only enter an incorrect code 4 times per kill. On the 5th incorrect code, the killer will forfeit his/her position in the game.
